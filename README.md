# Cadastro de Vendas

Exercício em JavaScript para cadastrar e exibir vendas em uma tabela.

## Funcionalidades

- Cadastrar vendedor e valor da venda
- Calcular automaticamente:
  - desconto de 10%
  - valor final
- Gerar ID automático
- Registrar data e hora
- Exibir os dados em uma tabela

## Ações disponíveis

- Remover uma venda específica
- Remover o último item
- Limpar toda a lista (com confirmação)

## Validações

- Nome obrigatório  
- Valor deve ser numérico  
- Valor maior que zero  

## Tecnologias

- HTML  
- CSS  
- JavaScript
